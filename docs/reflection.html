My Learning Journey

This module gave me a practical understanding of how to build and deploy professional digital projects using modern tools like Quarto, Git, and GitHub Pages. It bridged the gap between academic work and real-world software development practices.

Hands-On Learning

The hands-on learning approach played a key role in my development. By building a portfolio website from scratch, I learned not just the tools, but how to think like a developer — planning structure, writing clean code, and iterating based on feedback.

TipKey Insight
