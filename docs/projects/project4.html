Project Overview

This project compares two fundamentally different approaches to binary sentiment classification — fine-tuned BERT and zero-shot Llama3 — evaluated on the Stanford Sentiment Treebank v2 (SST-2) dataset. The business context is a streaming platform requiring real-time audience sentiment monitoring at scale.

Business Problem: Streaming platforms receive thousands of reviews daily. Manual sentiment analysis is neither scalable nor consistent. An automated system capable of classifying sentiment within hours of a title’s release is essential for editorial intervention before reputational damage occurs.

Success Criteria: ≥95% accuracy on negative sentiment · <4-hour classification latency · Deterministic, auditable outputs for regulatory compliance.

Dataset: Stanford Sentiment Treebank v2 (SST-2), part of the GLUE benchmark. Short movie-review sentences with binary labels (0 = negative, 1 = positive). A 150-sample subset (random state=42) was used for direct model comparison to ensure identical evaluation conditions for both models.

Model performance across all evaluation configurations.

Key Finding: BERT outperforms Llama3 across all metrics on the shared subset. However, Llama3’s zero-shot accuracy of 94.0% — achieved with no task-specific training data — demonstrates the impressive general capability of large generative models. BERT’s full validation accuracy (92.43%) falls marginally below the 95% production threshold, indicating additional training epochs are warranted before live deployment.

Ethical and Responsible AI

EU AI HLEG Trustworthy AI Framework — Key Considerations:

Criterion Implication for This Project
Human Agency Neither model operates autonomously — human editors review all alerts
Technical Robustness BERT is deterministic; Llama3 is non-deterministic (temperature > 0)
Privacy & Data Governance Ollama local deployment reduces third-party data exposure
Transparency BERT has no explainability; Llama3 can generate reasons (with hallucination risk)
Fairness SST-2 is predominantly English and Western — stratified subgroup testing required
Environmental Wellbeing BERT: ~0.5 kWh per 10K reviews vs Llama3: ~4–5 kWh
Accountability Fixed random seeds and error logs support partial auditability

Deployment Recommendation

Show cost comparison code 
cost_data = pd.DataFrame({

})

Recommended Hybrid Workflow:

  1. BERT classifies all incoming reviews within 30 minutes of submission
  2. An automated alert fires when negative sentiment exceeds 30% within 6 hours of a title’s release
  3. Llama3 generates on-demand natural-language explanations for ambiguous or flagged cases
  4. All final intervention decisions remain with human editorial staff

This hybrid approach balances BERT’s determinism and cost efficiency with Llama3’s qualitative explanation capability.

Key Insights

Finding Implication
BERT achieves 98.67% on 150-sample subset Strong task-specific performance with labelled data

Llama3 struggles with sarcasm and idioms Prompt engineering or few-shot examples could close this gap
