Project Overview

This project analyses over 61,000 consumer complaint records from the Consumer Financial Protection Bureau (CFPB) to identify inefficiencies in Bank of America’s complaint resolution process. Using NLP and an ensemble machine learning model, the system prioritises high-risk complaints that are likely to require monetary relief — enabling faster, more targeted resolution.

Business Problem: The existing First-In-First-Out (FIFO) triage system treats all complaints equally, causing delays in resolving high-risk financial cases and increasing operational costs.

Solution: A Fast-Track Routing System powered by a Voting Classifier (Gradient Boosting + Random Forest) to automatically flag complaints requiring urgent attention.

Analytical Approach

The project used a prescriptive analytics framework combining NLP and ensemble machine learning.

Key Pipeline Steps:

  • Data cleaning and preprocessing of 61,022 CFPB complaint records
  • Feature engineering: Days_to_Forward, cyclical month features (Month_Sin, Month_Cos)
  • Text analysis using TF-IDF (unigrams + bigrams, top 1,000 features) on complaint narratives
  • Binary classification target: complaints closed with monetary relief (positive class ~25%)
  • Voting Classifier combining Histogram Gradient Boosting and Random Forest
  • Class imbalance handled via probability calibration and threshold tuning
  • Evaluation metrics: F1-score and ROC-AUC (accuracy alone was insufficient due to imbalance)

Why Precision over Recall? With only ~25% of cases resulting in monetary relief, high precision ensures the specialist Senior Resolution Team is only deployed on genuinely high-risk cases — avoiding wasted resource allocation on false positives.

Key Insights

Finding Implication
Credit cards and savings accounts carry highest payout risk Prioritise these in routing logic
Complaint volume spikes correlate with increased risk probability Seasonal staffing adjustments needed
Longer complaint narratives predict higher likelihood of refund Text length is a meaningful feature
Web submissions resolve faster than postal/fax channels Shift resources toward digital response teams

Deployment

The model was deployed as an interactive Streamlit dashboard, providing:
